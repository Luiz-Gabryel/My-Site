## 🖥️ Portfólio Principal

Site pessoal desenvolvido com **HTML e CSS puro**, sem frameworks.

**Seções:**
- **Header** — apresentação com badge de disponibilidade e stack de tecnologias
- **Sobre Mim** — resumo do perfil e foco de estudos
- **Projetos** — cards com links para repositórios no GitHub
- **Contato** — links para telefone, e-mail e GitHub

**Tecnologias usadas:**
- HTML5 semântico
- CSS3 com variáveis, `clamp()`, Grid e Flexbox
- Google Fonts (Syne + Space Mono)
- Font Awesome (ícones)
- Deploy via Vercel
